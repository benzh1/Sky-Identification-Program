Sky Identification

This program takes in a video as an input file and then will output that video showing which areas of each frame of the video are part of the sky.
It will also output a probability map of the video showing how likely different sections of each frame are to be the sky.

To run the Python version, upload a video file named movie.avi and then run the Python file like normal
To run the C++ version, also upload a video file named movie.avi and then run the executable skyIdent through the terminal by running the command ./skyIdent
